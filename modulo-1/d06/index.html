Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utlizar em cada caso. Fiz isso usando a tag <abbr> que fornece uma descrição, funcionará como descrito anteriormente.

  • Parágrafo
  • Títulos
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas numeradas
  • Listas com demarcadores
